# Welcome to ServettiShop!

> Planning a party and want to make it more special? Look no further! We at ServettiShop rent napkins in multitude of qualities and colours.

This is a practice app for a simple shopfront of a napkin rental. The focus is more on practicing React rather than creating an examplary shop UI.
The UI has selection fields for napkin material, colour, and quantity, as well as fields for start and end date selection. The rent confirmation is just a pop up alert.
